# PlatinumRx Data Analyst Assignment

## Overview
This assignment demonstrates basic skills in SQL, Excel, and Python.

---

## SQL Tasks
- Created tables for Hotel and Clinic systems
- Used JOIN, GROUP BY, and aggregate functions
- Solved queries like:

  - Monthly billing
  - Revenue and profit calculation

---

## Excel Tasks

- Compared dates and time using formulas
- Identified tickets created and closed on same day and hour

---

## Python Tasks

- Removed duplicate characters from a string using loops
